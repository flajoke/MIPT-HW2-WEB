# LVD Store Frontend

Учебная пользовательская часть интернет-магазина светотехники для домашнего задания 3.

## Стек

- React
- React Router DOM
- Vite
- Mock-данные без подключения backend

## Реализованные страницы

- `/catalog` — каталог товаров с поиском, фильтрами и сортировкой
- `/product/:slug` — карточка товара с характеристиками и добавлением в корзину
- `/cart` — корзина с изменением количества и удалением товаров
- `/checkout` — оформление заказа с валидацией формы
- `/confirmation` — подтверждение заказа

## Запуск

```bash
cd frontend
npm install
npm run dev
```

Приложение будет доступно по адресу, который выведет Vite, обычно `http://localhost:5173`.

## Сборка

```bash
npm run build
npm run preview
```

Backend и админ-панель для этого этапа не требуются: интерфейс работает на статических mock-данных.
